## How to Use

1. When you encounter an error, grep this file for the error string:
   ```bash
   grep -i "YOUR_ERROR_STRING" myinsights/1nsights.md
   ```
2. If match found → read linked detail file → apply solution
3. If no match → debug → capture with `/myinsights [title]`

## Capture with

```
/myinsights [brief-title]
```
