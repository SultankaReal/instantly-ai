## Контекст для SPARC Planning

**Целевой сегмент:** SDR / Основатель стартапа (~50% пользователей)

**Core Value Proposition:**
> Единственный сервис в России с прогревом Яндекс.Почты и Mail.ru.
> Unlimited аккаунты. Flat-price. Первое письмо в inbox за 10 минут.

**Aha Moment:** Inbox Score 89%+ для Яндекс — видно сразу после регистрации

**Primary Growth Loop:** Community-Led (Telegram + vc.ru) → Trial → Конверсия → Реферал (+500 прогревов/мес)

**Paywall Strategy:** Показывается сразу после первого inbox-попадания (эмоция победы)

**Key Differentiators (Blue Ocean):**
- Прогрев Яндекс + Mail.ru (нет ни у кого в РФ в full-stack)
- AI Reply Agent (<5 мин) + unlimited accounts + flat ₽ price
- Multichannel (email + Telegram, v1.1)
- Поддержка на русском языке

**Pricing:**
- Старт: ₽1 990/мес
- Про: ₽4 990/мес
- Агентство: ₽9 990/мес

**Regulatory:** 38-ФЗ compliant — персонализированный B2B outreach

**Payments:** ЮКасса / СБП / T-Pay (без Stripe)
