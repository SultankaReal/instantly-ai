# Product Discovery Brief: Поток
## Instantly.ai аналог для рынка РФ

**Дата:** 2026-04-29 | **Режим:** DEEP | **Geography:** Россия
**Рабочий тайтл:** Поток (Cold Email Outreach SaaS)

---

## 1. One-Liner

> **Поток** — это Instantly.ai × прогрев Яндекс/Mail.ru × flat-price × поддержка на русском языке

Аналог Instantly.ai, адаптированный под реалии российского B2B-рынка: прогрев для отечественных почтовых провайдеров, оплата через ЮКасса/СБП/T-Pay, интерфейс на русском, соответствие 38-ФЗ.

## 3. Gap Analysis: Почему Instantly не работает в РФ

| Проблема | Влияние |
|----------|---------|
| Не прогревает Яндекс.Почту и Mail.ru | Критично — 60%+ B2B в РФ используют Яндекс/Mail.ru |
| Оплата только Visa/MC (заблокированы) | Невозможно подписаться из РФ |
| Интерфейс только на английском | Барьер для non-tech аудитории |
| Поддержка недоступна (только AI-бот) | Пользователи в РФ ждут русскоязычный саппорт |
| Нет учёта 38-ФЗ | Правовая неопределённость |
| Нет российской базы контактов | База 450M — глобальная, РФ-покрытие плохое |

## 5. Целевые сегменты (РФ)

### Сегмент A: B2B SDR / Основатель стартапа (~50%)
- **Functional Job:** "Отправлять 500+ персонализированных холодных писем в день на Яндекс/Mail.ru без спама"
- **Emotional Job:** "Чувствовать контроль над лидогенерацией без SDR-команды"
- **Trigger:** "Нам нужен pipeline, но бюджета на найм SDR нет"
- **Текущее решение:** Gmail/Яндекс вручную или Coldy.ai
- **Боль с текущим:** Нет AI-автоматизации, слабая персонализация

### Сегмент B: Аутрич-агентство (~30%)
- **Functional Job:** "Управлять outreach для 10–50 клиентов из одного кабинета без per-seat pricing"
- **Emotional Job:** "Масштабировать без найма SDR-команды"
- **Trigger:** "Клиент просит 50 qualified лидов за месяц"
- **Боль:** Нет multi-client агентского режима ни у одного РФ-решения

### Сегмент C: B2B SaaS маркетолог (~20%)
- **Functional Job:** "Интегрировать холодные рассылки в CRM-воронку (amoCRM/Bitrix24)"
- **Emotional Job:** "Иметь предсказуемый pipeline с dashboards"
- **Trigger:** "Директор по продажам поставил KPI 100 лидов/мес"
- **Боль:** Нет нативной интеграции с amoCRM/Bitrix24

## 9. Growth Strategy

**Primary Loop: Community-Led + Content (как у Instantly)**

```
YouTube/Telegram контент "Холодный аутрич в РФ"
    ↓
Привлекают SDR/основателей бесплатно (SEO + органика)
    ↓
Trial 7 дней → конвертируются в платящих
    ↓
Делятся результатами в Telegram/VC.ru → новые пользователи ↻
    ↓
Аффилиатная программа (20% recurring) → cannel CAC ~$0
```

**Moats:**
1. Warmup-сеть для Яндекс/Mail.ru (data moat, уникален для РФ)
2. Telegram-сообщество (switching cost)
3. All-in-one: leads + outreach + CRM в одном месте

## 10. Regulatory (РФ)

| Закон | Статус | Применение |
|-------|--------|------------|
| 38-ФЗ "О рекламе" | Действует | Холодные B2B письма законны если персонализированные (не массовая реклама) |
| 152-ФЗ "О персональных данных" | Действует | Хранение и обработка данных только на серверах в РФ |
| КоАП ст. 14.38 | Действует | Штраф за спам: 300К–1М руб. для компаний |

**Вывод:** Персонализированный B2B outreach законен при соблюдении условий персонализации.

## 12. Рекомендуемая стратегия входа

**Вариант A (Deliverability-first)** → приоритет для MVP:
- USP: "Единственный сервис в РФ с прогревом Яндекс + Mail.ru"
- Ориентируемся на Сегмент A (SDR/Основатели) — массовый и платёжеспособный

Multichannel (email + Telegram-рассылки) + warmup Яндекс + AI Reply Agent + flat-price + RU payments — ни один конкурент не имеет всего этого в одном продукте
